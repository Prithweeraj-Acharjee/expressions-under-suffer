## About

**Expressions Under Suffer** is the art portfolio of Prithweeraj Acharjee Porag: paintings, mixed-media collage, and writing on what painting means in the post-AI era. It gathers ongoing series alongside essays and research, presented as a living, contemporary body of work.

**Live at:** [expressions-under-suffer.vercel.app](https://expressions-under-suffer.vercel.app)

---

## Series

- **Central Face**
- **Freedom Under** (collage on the gaze)
- **Trouble in Head**
- **Isolated**
- **Fighting Bugs**
- **Underneath, Remain**

---

## Built With

Next.js, TypeScript, Tailwind CSS, and Framer Motion.

---

## Run Locally

```bash
git clone https://github.com/Prithweeraj-Acharjee/expressions-under-suffer.git
cd expressions-under-suffer
npm install
npm run dev
```

Then open `http://localhost:3000`.
